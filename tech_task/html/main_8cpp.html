Detailed Description

The file contains an algorithm that searches for a point on a polyline that is closest to a given point.

The file contains classes that characterize geometric objects such as point, line segment and vector. Function dist_point_segment calculates the shortest distance from a point to a segment and calculates the point of the segment at which this distance is reached. Function main contains processing of input data for correctness, as well as most of the remaining calculations to solve the problem.

Searches for a point on a polyline that is closest to a given point and outputs the number of such points in the console, as well as the points themselves, indicating the segment number. Processes input data for correctness. It is considered that if there is at least one point in a polyline, then the shortest distance from a given point to the polyline does not exceed the distance from a given point to this one point.
