Detailed Description

Class MyVector refers to the vector in three-dimensional space.

A vector is defined by three coordinates x, y, z. A vector can be specified using start and end points, by 3 coordinates of vector or by segment, where the vector points from the first endpoint of segment to the second endpoint. A vector can be multiplied by a scalar. Can be calculated the scalar and vector product of a vector and a given vector. Vector length can be calculated. A given point can be moved by a vector.
